## Overview
This repository contains the complete solution for the SHL Grammar Scoring challenge. The approach implements an end-to-end audio processing and deep learning pipeline to predict continuous grammar scores from spoken audio samples.

The solution is designed for reproducibility, clarity, and efficient execution, with GPU acceleration enabled when available.
